== Description ==
Keep your user data up to date by prompting the user to confirm their email address or enter a new email address every X amount of logins or X amount of days.

== Installation ==
Installing the Confirm data plugin is as easy as clicking "Install Now" from your dashboard's plugin page or use the download button, top right.

== Frequently Asked Questions ==
<h4>Where do I find the link to the settings for Confirm Data Plugin?</h4>
Look for Confirm Data Plugin under Settings or get it at /wp-admin/options-general.php?page=wec_config

<h4>Can I change the Custom CSS?</h4>
Of course, feel free to overwrite the CSS on the settings page, whatever you do there should not overwrite your theme Custom CSS

<h4>What if I do not want to confirm a users details based on X days?</h4>
Set the Check user email address each X days to something like 10000, default is 10

<h4>What if I do not want to confirm a users details based on X logins?</h4>
Set the Check user email address each X logins to something like 10000, default is 10
